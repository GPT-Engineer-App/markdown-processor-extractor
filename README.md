# markdown-processor-extractor

A markdown processor component that extracts, labels, and creates separate output(s) for any code blocks, tables, and mermaid diagrams that are found in markdown input.

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/markdown-processor-extractor.git
cd markdown-processor-extractor

```


This will run a dev server with auto reloading and an instant preview.
